# Stella Style sample contract

This directory contains one fictional Mutual Services Framework Agreement rendered with the curated **Stella Style** set. The document demonstrates formatting only and is not legal advice.

## Curated style set

Stella Style is intended as a strong, neutral starting point for a new legal document:

- A4 page size with 25.4 mm margins
- Arial 10 pt body text, justified
- black-and-white presentation with no firm-specific branding
- reference-derived C-level hierarchy: `1` bold, then regular-weight `1.1`, followed by `(a)`, `(i)`, and `(A)`
- named styles for title, subtitle, parties, recitals, agreed terms, clause headings, ordinary `1.1` clause paragraphs, nested clause paragraphs, definitions, tables, schedules, footnotes, footer text, and hyperlinks
- paragraph spacing and hanging indents designed for contracts rather than general correspondence

## Research references

The preset was checked against neutral, internationally used drafting patterns, including oneNDA and World Bank standard-form documents. Their logos, colours, wording, and distinctive branding are not included. These references informed the comparison work only; Stella Style is the sole curated preset shipped here.

- oneNDA: https://www.onenda.org/products/onenda
- World Bank procurement framework: https://www.worldbank.org/ext/en/what-we-do/project-procurement/framework
